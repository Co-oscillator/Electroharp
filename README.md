# Electroharp v1.3
Firmware for 2.8 and 3.5 hardware targets.

Tested on:
Hosyond 3.5'' 320x480 Resistive Touch Screen ESP32 (Amazon)
Hosyond 2.8'' 240x320 Resistive Touch Screen ESP32 (Amazon)
Unbranded 2.8'' 240x320 Resistive Touch Screen ESP32 (AliExpress)

Designed to work with small wired speakers connected to the Cheap Yellow Display's speaker port, or to send audio to a bluetooth audio device with acceptably low latency.
